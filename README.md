# Autoservice

Django проект для управління автосервісом.

## Структура проекту

```
autoservice/          # Головні налаштування проекту
├── settings.py       # Налаштування Django
├── urls.py           # Головні маршрути
├── views.py          # Представлення
├── wsgi.py           # WSGI конфігурація
└── asgi.py           # ASGI конфігурація

blog/                 # Додаток для блогу
├── models.py         # Моделі даних
├── views.py          # Представлення
├── urls.py           # Маршрути додатку
├── admin.py          # Адміністративна панель
└── migrations/       # Міграції бази даних

templates/            # HTML шаблони
└── basic/
    ├── base.html     # Базовий шаблон
    └── index.html    # Домашня сторінка

static/               # Статичні файли
└── css/
    └── main.css      # Основні стилі

manage.py             # Утиліта управління проектом
db.sqlite3            # База даних SQLite
```

## Встановлення

1. Клонуйте репозиторій
2. Встановіть залежності:
```bash
pip install django
```

3. Виконайте міграції:
```bash
python manage.py migrate
```

## Запуск проекту

```bash
python manage.py runserver
```

Проект буде доступний за адресою `http://127.0.0.1:8000/`

## Використані технології

- **Django** - веб-фреймворк
- **SQLite** - база даних
- **HTML/CSS** - фронтенд
